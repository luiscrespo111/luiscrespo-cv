Exportar CV Actualizado

Al presionar el botón se genera y descarga el archivo index.html con todos los datos actualizados. Luego reemplazalo en tu repositorio GitHub.

📋 Cómo subir a GitHub

Seguí estos pasos para publicar los cambios en tu CV online:

1. Descargá el index.html generado
2. Abrí tu repositorio en GitHub:
   github.com/luiscrespo111/luiscrespo-cv
3. Hacé clic en index.html → Edit (lápiz)
4. Borrá todo el contenido y pegá el nuevo
5. Commit changes → "Actualizar CV"
6. GitHub Pages publica en ~30 segundos

O con Git desde la terminal:

git add index.html
git commit -m "Actualizar CV"
git push origin main
